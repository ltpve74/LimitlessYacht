## 3. Target Countries

## 9. Conversion Tracking

GTM (GTM-NN8V25BR) is already installed. Set up the following conversion events via GTM before launching ads:

| Conversion Action | Type | How to Trigger |
|-------------------|------|----------------|
| **Form Enquiry** | Primary | Element Visibility trigger — fires when `#form-success` becomes visible on the page |
| **WhatsApp Click** | Primary | Click trigger — URL contains `wa.me` |
| **Enquire Now Button** | Secondary | Click trigger — `.btn-primary` links pointing to `#contact` |
| **Pricing Section View** | Secondary | Scroll depth or element visibility on `#pricing` — indicates high intent |

**Setup steps:**
1. In GTM, create a Google Ads Conversion Linker tag (fires on All Pages)
2. Create a Google Ads Conversion tag for each action above
3. Link GTM container to Google Ads account under Tools → Conversions → Google Tag Manager
4. Enable **Enhanced Conversions** — use the email field from the contact form for better attribution

## 11. Budget Recommendation

Given charter rates of €3,300–€26,000, a single booking from ads covers months of ad spend.

| Scenario | Daily Budget | Monthly | Est. Enquiries/Month | Notes |
|----------|-------------|---------|----------------------|-------|
| **Starter** | €25–35/day | €750–1,050 | 6–12 | Good for testing and initial data |
| **Growth** | €50–80/day | €1,500–2,400 | 15–25 | Recommended starting point |
| **Aggressive** | €120–150/day | €3,600–4,500 | 40–60 | Scale once conversion rate is proven |

## 12. Ad Schedule & Seasonality

**Daily schedule:** Run 24/7 initially, then optimize after 2–3 weeks. Likely strongest: 9am–10pm local time.

**Seasonal peaks — increase budget during:**
- **January–April:** People planning summer Mallorca holidays (highest search intent)
- **May–June:** Last-minute peak-season bookings
- **September–October:** Planning autumn/shoulder season trips

**Off-peak note:** Shoulder season (Sep–May) pricing is €3,300/day. Ads can emphasise the value vs. peak pricing. Run year-round — even winter generates enquiries for the following summer.
